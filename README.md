# PWA Portfolio Project

This repository contains multiple portfolio implementations, showcasing modern web development techniques with Progressive Web App (PWA) features.

## Projects Included

- **my-portfolio**: A traditional Next.js portfolio implementation
- **pwa_portfolio_01**: Enhanced portfolio with AI chatbot, modern UI, and advanced features

## Features

- Responsive design
- Modern UI/UX
- AI-powered chatbot
- Contact form with email functionality
- Social media integration
- PWA capabilities

## Technologies Used

- Next.js 14+
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React
- Google Generative AI
